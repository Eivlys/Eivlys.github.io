报错处理

EventEmitter memory leak detected

解决方法：启动文件中添加语句‘process.setMaxListeners(0);’可关闭报警，但可能仍存在监听过载或内存溢出的问题

Error: Can’t set headers after they are sent.
可能原因：res.end()之后又res.send()，或者二次回调，检查代码。
参考解决方法：
https://github.com/expressjs/express/issues/751；
http://stackoverflow.com/questions/7042340/error-cant-set-headers-after-they-are-sent-to-the-client
