可能出现的问题

1.出现 hexo:command not found
解决：
[1].npm没有安装：
安装nvm → 安装npm （nvm install 0.12）→ 看版本（node –version）

[2].没有指定npm版本：
eg：
nvm ls
nvm use 0.12
nvm alias default 0.12

2.修改文件后页面没有更新
解决：清理缓存并重新生成文件

3.无法添加about
解决：
编辑主题配置文件_config.yml

然后执行new page命令

编辑在hexo\source\里的index.md

也可以在hexo\source\下手动建立about和index.md

4.在Windows环境下部署
显示Error: spawn git ENOENT：:添加git的bin文件所在路径到PATH环境变量

其他
